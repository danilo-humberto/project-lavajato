# Lava+ Landing Page

Primeira versão visual estática de uma landing page para lavajato automotivo.

## Tecnologias

- React
- Vite
- TypeScript
- TailwindCSS
- GSAP instalado para uso futuro, sem animações nesta etapa

## Comandos

```bash
npm install
npm run dev
npm run build
```

## Escopo atual

- Landing page responsiva e estática.
- Componentes separados por seção.
- Sem backend, admin, kanban, localStorage, cálculo de preço ou animações.
